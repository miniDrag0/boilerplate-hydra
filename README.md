# Hydra Electron Boilerplate

This is an Electron implementation of the Hydra UI.

## How to Run

1.  Make sure you have Node.js installed.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the application:
    ```bash
    npm start
    ```

## Structure

-   `main.js`: The Main Process entry point. Creates the application window.
-   `index.html`: The UI layout.
-   `styles.css`: The styling matching the screenshot.
